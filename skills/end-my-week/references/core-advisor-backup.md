# Advisor 备份系统

## 目的

Advisor 文件存储完整的本周复盘信息，用于：
1. 记录周计划与实际执行的偏差分析
2. 供下周 `start-my-week` 参考
3. 保存需要进入 monthBacklogs 的任务

## 文件路径

```typescript
const ADVISOR_PATH = (weekId: string) => 
  `docs/dashboard/advisor/${weekId}-end.md`
```
