# VueUse Combo Skill Requirements

This combo skill adapts the official VueUse skills with additional focus on:

## Primary Focus Areas

1. **TypeScript Usage**: Emphasize TypeScript patterns and type safety in all examples
2. **Composition API**: Focus on Vue 3 Composition API usage patterns
3. **Real-world Examples**: Include practical, production-ready code snippets
4. **Performance Considerations**: Highlight performance implications and best practices

## Customizations Needed

- Add TypeScript generic examples for functions with generics
- Include error handling patterns for reactive functions
- Add integration examples with Pinia and Vue Router
- Emphasize tree-shaking and bundle size considerations

## Skippable Content

- Basic Vue 3 setup tutorials
- Installation and configuration guides
- Non-TypeScript JavaScript examples
- Legacy Options API usage
